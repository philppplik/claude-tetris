The coupling

How the pause magic works

Claude Code hooks drive a tiny signal file. No polling, no lag.

01

Hook fires

A UserPromptSubmit or Stop hook in Claude Code triggers when you start or finish.

02

Signal writes

tetris-signal.mjs play|pause writes a single state.json atomically.

03

Game reacts

The TUI watches the file via fs.watch → freezes on "Claude is done", resumes instantly.

Get started

Install in 10 seconds

Double-click for the easy path, or use the command line.

🪟 Easy (Windows)

  1. Double-click install.bat
  2. Hooks install automatically (your settings.json is backed up)
  3. Open the split pane when prompted

To uninstall: double-click uninstall.bat.

Command line 
npm run install:hooks
npm run launch        # open split pane
Slash command 
/tetris   # launch from inside Claude Code
